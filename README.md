# Specification

- PHP >= 8.1
- NodeJS >= 18

# Installation

```shell
git clone https://github.com/ahmakbardev/NEW_ARTIKNESIA
```

```shell
cd NEW_ARTIKNESIA
```

```shell
cp .env.example .env
```

```shell
composer install
```

```shell
npm install
```

# Running Backend

```shell
php artisan migrate
```

```shell
php artisan key:generate
```

```shell
php artisan serve
```

# Running Frontend

```shell
npm run dev
```

# Build Frontend

```shell
npm run build
```
